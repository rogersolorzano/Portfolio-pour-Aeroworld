# Portfolio de professionnel de la data

## Description
Ce projet a consisté à concevoir un portfolio en ligne personnalisé qui met en valeur mes compétences, expériences et réalisations dans le domaine de la data. L’objectif était d’améliorer mon employabilité en présentant un support clair et professionnel, spécifiquement adapté aux attentes des recruteurs d’AéroWorld, notamment autour des thématiques de management et d’innovation.

## Objectifs
Valoriser mon parcours et mes compétences techniques pertinentes pour AéroWorld (analyse de données, modélisation, visualisation, gestion de projet).
Présenter mes projets clés avec un focus sur les résultats concrets et les usages pratiques.
Rendre le portfolio accessible, attrayant et facile à naviguer pour capter efficacement l’attention des recruteurs.
Démontrer ma capacité à structurer une communication professionnelle autour de mes travaux dans un contexte d’innovation.

## Outils utilisés
Générateur de site statique (GitHub Pages) pour héberger le portfolio en ligne.
Outils de mock-up (Figma) pour concevoir l’interface et structurer le contenu.
Technologies web simples (HTML, CSS) pour personnaliser le design et assurer la responsivité.
Intégration de liens vers des projets hébergés sur GitHub, notebooks, et dashboards interactifs.

## Livrables Aéroworld

### 1. Document d’analyse du besoin client
Objectifs pour le recruteur : Montrer ma compréhension complète des attentes et contraintes du client.

Description : Document détaillant les besoins exprimés par AéroWorld, leurs enjeux et les spécifications initiales.
 
### 2. Cahier des charges du projet
Objectifs pour le recruteur : Présenter une synthèse claire des objectifs, périmètre et livrables du projet.

Description : Document formalisant les attentes, les indicateurs clés et le planning prévisionnel.

### 3. Diagramme de Gantt (réalisé avec Power BI)
Objectifs pour le recruteur : Visualiser l’organisation temporelle du projet et la planification des tâches.

Description : Diagramme interactif présentant les jalons, les dépendances et l’avancement.

### 4. Dashboard Veille (Power BI)
Objectifs pour le recruteur : Illustrer ma capacité à créer des tableaux de bord dynamiques de suivi et analyse.

Description : Dashboard de veille sectorielle permettant une surveillance en temps réel des indicateurs clés.

### 5. Dashboard Profil (Tableau)
Objectifs pour le recruteur : Montrer mon expertise dans l’utilisation de Tableau pour concevoir des visualisations claires et interactives.

Description : Tableau de bord orienté profil utilisateur, synthétisant des données de compétences et performances.

### 6 Vidéo de formation
Objectifs pour le recruteur : Mettre en avant mes aptitudes pédagogiques et ma capacité à transmettre mes connaissances.

Description : Vidéo expliquant l’utilisation ou la réalisation d’un livrable clé du projet (ex. création d’un dashboard).

### 7. Documentation sur création de graphique avec avec Tableau
Objectifs pour le recruteur : Fournir un support clair et visuel pour accompagner les équipier à leur prise en main de l'outil Tabelau.

Description : guide illustrant la création d'un graphique avec Tableau, depuis l'importation des données.
